### Development Stack

**Gancio** is built with following technologies:

- [Nuxt.js](https://nuxtjs.org/)
- Vue.js
- Express
- Sequelize
- Element.ui

### Testing on your own machine

2. Download source
```bash
git clone https://git.lattuga.net/cisti/gancio
```

3. Install dependencies
```bash
yarn
```

4. Hacking
```bash
yarn dev
```
